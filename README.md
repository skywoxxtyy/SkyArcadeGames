## О проекте

**Sky Arcade Games** — это сборник классических аркадных игр, написанный на C# с использованием Windows Forms. Проект создан с нуля с упором на красивый UI в стиле **dark cyberpunk**, плавную анимацию и приятный игровой процесс.

Все игры объединены общим меню, единой системой сохранения статистики и достижений.

---

## Игры

### 🟦 Тетрис
Классический тетрис с современной визуальной подачей.
- 15+ уровней с нарастающей скоростью
- Система очков с бонусами за несколько линий
- Предпросмотр следующей фигуры

### 💣 Сапёр
Классическая игра на поиск мин.
- 3 уровня сложности: Новичок (9×9), Средний (16×16), Эксперт (30×16)
- Таймер и счётчик мин
- Рекорды времени для каждой сложности
- Серии побед

### 🏎️ Гонки
Аркадные гонки с видом сверху.
- **4 карты:** Город, Пустыня, Ночь, Лёд — каждая с уникальной визуализацией
- **5 уровней сложности** (от Новичка до Ада)
- Система жизней, щит, нитро-бустер
- Монеты, комбо-множитель, мини-карта
- Детализированные машины с анимацией

### 🐍 Змейка
Классическая змейка с несколькими режимами игры.
- **4 карты:** Классика, Неон, Лес, Космос
- **4 режима:** Классика, Без стен, Два яблока, Препятствия
- Градиентное тело змейки, анимированные глаза
- Частицы при поедании еды
- Комбо-система и нарастание скорости

## Установка и запуск

### Требования
- Windows 10 / 11
- [.NET 6.0+](https://dotnet.microsoft.com/download) *(или используй Self-Contained сборку — не нужен .NET)*

### Запуск из исходников
```bash
git clone https://github.com/skywoxxtyyy/sky-arcade-games.git
cd sky-arcade-games
dotnet run --project ArcadeHub
```

### Готовый .exe
Скачай последний релиз на странице [Releases](../../releases) и запусти `ArcadeHub.exe`.

## Управление

| Игра | Управление |
|------|-----------|
| **Тетрис** | `←` `→` — движение, `↑` — поворот, `↓` — ускорение, `Space` — сброс |
| **Сапёр** | `ЛКМ` — открыть клетку, `ПКМ` — поставить флаг |
| **Гонки** | `←` `→` — руль, `↑` `↓` — газ/тормоз, `Space` — нитро, `P` — пауза |
| **Змейка** | `←` `→` `↑` `↓` или `WASD` — движение, `P` — пауза, `Space` — новая игра |

## Система достижений

В игре есть **25+ достижений** по всем играм:

- 🏁 Гонки — дистанция, монеты, скорость
- 🐍 Змейка — длина, яблоки, серии
- 🟦 Тетрис — линии, уровни, рекорды
- 💣 Сапёр — победы, серии, сложность

Достижения отображаются в виде тост-уведомлений прямо во время игры и хранятся в общем журнале.

---

## Статистика

Каждая игра ведёт собственную статистику, которая отображается прямо в боковой панели во время игры:

- Рекорды очков и времени
- Общее количество игр, монет, яблок, линий
- Суммарное игровое время
- Текущие и максимальные серии
